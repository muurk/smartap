## Why AGPL-3.0?

The AGPL-3.0 was chosen because:

1. **Network Copyleft** - If you run a modified version as a network service, you must share your source code. This ensures improvements benefit everyone.

2. **Strong Copyleft** - Derivative works must remain open source under the same license.

3. **Community-Oriented** - Encourages contributions back to the project rather than proprietary forks.
